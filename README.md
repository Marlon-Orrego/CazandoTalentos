# Tech And Solve Recruitment Portal
## Descripción
El Portal de Reclutamiento de Tech And Solve es una plataforma web diseñada para simplificar y agilizar el proceso de reclutamiento de talento humano. Permite a los equipos de Recursos Humanos de Tech And Solve y a los candidatos interactuar de manera eficiente y automatizada a lo largo de todo el proceso de selección.
